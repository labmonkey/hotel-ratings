### Installation
1. Copy files into preferred PHP server public folder
2. That's it! It's ready to use.
### Developers
1. `composer update`
2. `cd views`
3. `npm install`
4. `bower install`
5. `gulp` - run each time

##### Database
In order to update database simply run `sudo sh includes/model/commands/update-db.sh`
